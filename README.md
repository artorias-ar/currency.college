Currency College - Конвертер валют
Небольшая программа для конвертации валют с актуальными курсами.

Функциональность
- Конвертация между популярными валютами (USD, EUR, RUB, KZT и др.)
- Получение актуальных курсов с внешнего API
- Простой и понятный веб-интерфейс

Технологии
- **Python**
- **API**
- **Деплой** Localhost
